## Complete Workflow: Creating a Blog Post

### Step 1: Access Blog Module
1. Navigate to **Content & SEO** in main menu
2. Click **Write Blog**
3. Click **"Create New Blog"** button

### Step 2: Enter Basic Information
1. **Enter Title**: Type compelling, SEO-friendly title
2. **Add Excerpt**: Write brief summary (2-3 sentences)
3. **Select Category**: Choose from existing or create new
4. **Add Tags**: Input relevant keywords

### Step 3: Write Content
1. **Use Rich Editor**: Type or paste content
2. **Format Text**: Apply headings, bold, lists
3. **Add Links**: Insert internal/external links
4. **Insert Media**: Add images, videos, infographics
5. **Check Readability**: Use built-in analyzer

### Step 4: Optimize for SEO
1. **Set Focus Keyword**: Primary search term
2. **Edit Meta Title**: Optimized for search engines
3. **Write Meta Description**: Compelling summary
4. **Customize URL Slug**: Short, keyword-rich URL
5. **Add Alt Text**: For all images

### Step 5: Add Visual Elements
1. **Upload Featured Image**: Main blog image
2. **Insert In-Content Images**: Support your text
3. **Add Captions**: Describe images
4. **Optimize Sizes**: Ensure fast loading

### Step 6: Configure Settings
1. **Author Attribution**: Select blog author
2. **Comments Setting**: Enable/disable comments
3. **Social Sharing**: Configure share buttons
4. **Related Posts**: Auto or manual selection

### Step 7: Review and Preview
1. **Click Preview**: See live version
2. **Check Desktop View**: Full-width layout
3. **Check Mobile View**: Responsive design
4. **Test Links**: Verify all hyperlinks
5. **Proofread**: Final grammar/spelling check

### Step 8: Publish or Schedule
**Option A - Immediate Publish**:
1. Click **"Publish"** button
2. Confirm publication
3. Blog goes live instantly

**Option B - Schedule**:
1. Click **"Schedule"** option
2. Select date from calendar
3. Choose time
4. Confirm schedule
5. Blog publishes automatically

### Step 9: Post-Publish Actions
1. **Share on Social Media**: Auto-post to platforms
2. **Include in Newsletter**: Add to email campaign
3. **Monitor Analytics**: Track views, engagement
4. **Respond to Comments**: Engage with readers
5. **Update if Needed**: Keep content current

## SEO Best Practices

### On-Page SEO Elements

#### 1. Title Optimization
- Include primary keyword near beginning
- Keep under 60 characters
- Make it compelling and clear
- Use numbers when relevant ("5 Tips...", "10 Ways...")

#### 2. Meta Description
- 150-160 characters
- Include focus keyword
- Call-to-action phrase
- Unique for each post

#### 3. Content Structure
- Use H1 for title only
- H2 for main sections
- H3 for subsections
- Short paragraphs (2-3 sentences)
- Bullet points and lists

#### 4. Keyword Usage
- Natural integration (no stuffing)
- 1-2% keyword density
- Variations and synonyms
- Long-tail keywords

#### 5. Internal Linking
- Link to related blog posts
- Link to service pages
- Use descriptive anchor text
- 2-5 internal links per post

#### 6. External Linking
- Link to authoritative sources
- Medical journals and studies
- Government health sites
- Industry experts

#### 7. Image Optimization
- Compress images (under 100KB)
- Descriptive filenames
- Alt text with keywords
- Proper dimensions

#### 8. Readability
- Simple language (8th grade level)
- Active voice
- Transition words
- Varied sentence length

#### 5. Seasonal Health Tips
- Flu season prevention
- Summer safety
- Allergy management
- Holiday wellness

#### 6. FAQ Posts
- Common patient questions
- Myth-busting articles
- Procedure explanations
- Cost transparency

### Writing Style Guidelines

**Tone**:
- Professional yet approachable
- Empathetic and understanding
- Authoritative but not condescending
- Clear and jargon-free

**Voice**:
- Active voice preferred
- Second person ("you") for patient-focused content
- First person plural ("we") for clinic news
- Consistent across all posts

**Length**:
- Minimum 300 words
- Ideal: 800-1,500 words
- Comprehensive guides: 2,000+ words
- Quality over quantity

## Troubleshooting

### Common Issues and Solutions

#### Issue 1: Images Not Uploading
**Possible Causes**:
- File too large
- Unsupported format
- Slow internet connection

**Solutions**:
- Compress image before upload
- Convert to JPG or PNG
- Check internet speed
- Try drag-and-drop method

#### Issue 2: SEO Score Low
**Possible Causes**:
- Missing meta description
- Keyword not in title
- Short content length
- No internal links

**Solutions**:
- Complete all SEO fields
- Expand content to 500+ words
- Add 2-3 internal links
- Use focus keyword naturally

#### Issue 3: Preview Not Loading
**Possible Causes**:
- Browser cache issue
- JavaScript error
- Server timeout

**Solutions**:
- Clear browser cache
- Try incognito/private mode
- Refresh page and try again
- Contact support if persists

#### Issue 4: Cannot Schedule Post
**Possible Causes**:
- Past date selected
- Timezone mismatch
- Permission restrictions

**Solutions**:
- Select future date only
- Verify timezone settings
- Check user permissions
- Admin approval may be needed

## Compliance and Legal Considerations

### Medical Content Accuracy
- **Review Process**: Medical professional review required
- **Citations**: Link to peer-reviewed sources
- **Disclaimers**: Include medical disclaimer
- **Date Stamps**: Show last updated date

- **Consent**: Written consent for patient stories
- **Anonymization**: Remove identifying details
- **Data Protection**: Secure storage of drafts

- **Alt Text**: All images described
- **Color Contrast**: Readable text
- **Keyboard Navigation**: Full accessibility
